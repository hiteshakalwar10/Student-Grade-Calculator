# Student Grade Calculator

A beginner-friendly Python project that reads student marks from a text file, calculates the average marks, assigns grades, and displays the results.

## Features

- Read student data from a .txt file
- Calculate average marks
- Assign grades (A, B, C, D, F)
- Uses functions and `return`
- Demonstrates file handling
- Includes exception handling for missing files

## Technologies Used

- Python 3
- Google Colab
